4.6.4. Thermal Expansion of Copper Case Study

4.6.4.1.

Background and Data
Description of the Data The response variable for this data set is the coefficient of thermal expansion for copper. The predictor variable is temperature in degrees kelvin. There were 236 data points collected.

These data were provided by the NIST scientist Thomas Hahn.

Software The analyses used in this case study can be generated using both Dataplot code and R code.
