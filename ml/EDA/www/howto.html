How to use the Engineering Statistics Handbook
Helpful hint Use your browser 'Back' button to return to the page you have just left.
Use the graphic icon on the tool bar to go 'home' Home. A tool bar appears at the top and bottom of every page. The the graphic icon on the toolbar takes you to the main Contents page, from which each chapter can be reached, as well as Tools & Aids, and a detailed Table of Contents for the entire handbook.
Use links to move to different levels in a chapter. Chapter navigation. At the top of each page, under the toolbar, there are a set of links that identify the chapter; chapter section; and chapter sub-section that you are currently in. These links can be used to move between levels within the chapter and back to the high level outline for the entire chapter.
Use arrows to move betwen pages Page navigation. At the top and the bottom of every page there are navigation arrows to take you backward or forward one page. This navigational tool can be used to step through the entire Handbook.
Statistical software used to analyze example data Software. Two software packages are used to analyze example data in the Handbook, Dataplot and R. Both packages are free and downloadable. Many links to actual Dataplot code and R code appear throughout the Handbook.
Download Dataplot and try the Handbook examples yourself Dataplot Software.Use Dataplot to analyze Handbook examples yourself. Many parts of the book feature case studies or short examples with computations from Dataplot. The Dataplot software can be downloaded from the Dataplot home page and installed on your system so you can run the software with sample data from the Handbook data or your own data. The full documentation for Dataplot is also accessible from the Dataplot home page.
Activate Dataplot from a case study Running Dataplot. Dataplot is configured to automatically activate when you click any hyperlink under 'Run this Example Yourself' from a Handbook case study - provided your Web browser has been set up to call up Dataplot. Dataplot then brings up a separate screen which is divided into four windows: 
  1. Upper left - Feedback and computational output
  2. Lower left - Dataplot commands as they are executed
  3. Upper right - Graphical output
  4. Lower right - Spreadsheet of data
At the bottom of the screen is a wide, rectangular window where you can enter Dataplot commands to be executed.

Once Dataplot is running it should be possible to click on the hyperlinks for each step in case study to continue the analysis without exiting the software.

To exit Dataplot when finished with an analysis, click on the Exit button on the menu bar.
Use Tools & Aids to find the place in the handbook where your problem is matched to a solution Problem/Solution Matching. If your problem is an engineering or scientific problem, try to match the questions you are interested in to the Handbook's list of: 
  • Engineering Questions.
If you have already translated your problem into a statistical framework, look under the following aids for guidance:
  • Gallery of Graphical Techniques 
  • Gallery of Analytical Techniques 
If you would like guidance on your problem via a real-life situation, try looking at the:
  • Gallery of Case Studies.
All of these items can be found under Tools & Aids on the Home page.
For in-depth outline Go to the Detailed Table of Contents from the Home page.
For a specific topic Click on 'Search Handbook' under Tools & Aids on the Home page.
