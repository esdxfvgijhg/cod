Following is a list of all files in the distribution and a brief description of each, starting with the files installed as part of VisualWorks.  Parcels typically come in two files, the parcel file (.pcl) and the parcel source file (.pst).  Parcel source files are omitted from the list since the description is provided for the associated parcel file.
