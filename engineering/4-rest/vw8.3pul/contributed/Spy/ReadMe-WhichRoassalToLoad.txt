Roassal2 is the new version of Roassal.  The Spy tool in this release still uses the old version of Roassal.

 - Loading Roassal2 gives the powerfull visualisation engine.  Loading Roassal2-Dev loads it plus its tests and examples.

	See https://www.youtube.com/watch?v=FxqLksMOQFg for a demo.

 - Loading the Spy coverage tool loads the old version of Roassal.

Do not have Roassal2 and Roassal loaded in the same image.
