The /contributed directory is a renamed version of what was formerly called /goodies, with a flattened hierarchy. 

"Contributed" code provides a wide variety of extensions to VisualWorks, including frameworks for various programming tasks, additional database support for Store, tools, free versions of third-party software, and so on. Some contributions include documentation provided by their developers; many do not.

See fileList.txt for one-line descriptions of the included files.
