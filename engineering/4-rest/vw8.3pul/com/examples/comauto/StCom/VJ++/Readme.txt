COM Connect 1.1
Microsoft Visual J++ 1.1 Example Client
---------------------------------------

The VJstCom directory contains a Microsoft Visual J++ 1.1
project for a client Applet that uses the VisualWorks
Smalltalk Commander COM Server example provided with 
COM Connect 1.1.

The interesting files in the VJstCom directory are:

VJStCom.dsw: The Microsoft Visual J++ 1.1 project workspace.
VJStCom.java: The main Java source file.
VJStCom.html: An HTML file to run the applet.

If you run VJStCom.html by itelf (not from Microsoft Visual J++)
and the applet does not run, your system may not be configured
properly. This can happen even if the applet runs when 
Microsoft Visual J++ lauches your web browser.

Double-click on VJStCom.dsw to start Microsoft Visual J++.
